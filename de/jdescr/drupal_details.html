Drupal/MySQL jail

For installing Drupal7 with MySQL backend jail use bsdconfig cbsd and Repository submenu, or just follow this steps:

			% cbsd repo action=get sources=img name=drupal

or

			% cbsd repo action=get sources=img name=drupal ver=10.1

If you hosted on another ( non 10.1 ) FreeBSD version.

Execute //wf_imghelper_ssi.html">imghelper that asks you for some questions required for the installation of the jail:

% cbsd imghelper jname=drupal bootstrap=/usr/jails/jails-system/drupal/bin/bootstrap.sh

where: /usr/jails/ is workdir for CBSD.

After entering FQDN of instance, Install procedure and starting jail the clean Drupal installation will be available via http://FQDN/
