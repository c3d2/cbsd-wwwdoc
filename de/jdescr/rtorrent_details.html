Rtorrent for FreeBSD CBSD Jail

Rtorrent jail

To install rtorrent cbsd jail use bsdconfig cbsd and Repository submenu, or just follow this steps:

			% cbsd repo action=get sources=img name=rtorrent

or

			% cbsd repo action=get sources=img name=rtorrent ver=10.1

If you hosted on another ( non 10.1 ) FreeBSD version.

Execute //wf_imghelper_ssi.html">imghelper that asks you for some questions required for the installation of the jail:

% cbsd imghelper jname=rtorrent bootstrap=/usr/jails/jails-system/rtorrent/bin/bootstrap.sh

where: /usr/jails/ is workdir for CBSD.

If all goes well, when you start the jail:

			% cbsd jstart rtorrent

Please note: all torrent will be stored in the /home/web/downloads/ directory inside the jail. You can share this dir with master node or other jail via fstab file.
