Bacula jail

To install bacula cbsd jail use bsdconfig cbsd and Repository submenu, or just follow this steps:

			% cbsd repo action=get sources=img name=bacula

After downloading images and set correct ip address of jail, run install script who will ask you for a password for root user in MySQL, password of bacula user in MySQL and the name of the resource will be available via http://FQDN:

Execute //wf_imghelper_ssi.html">imghelper that asks you for some questions required for the installation of the jail:

% cbsd imghelper jname=bacula bootstrap=/usr/jails/jails-system/bacula/bin/bootstrap.sh

where: /usr/jails/ is workdir for CBSD.

If all goes well, when you start the jail:

			% cbsd jstart bacula
