NOC Project FreeBSD jail image

NOC Project is the scalable, high-performance and open-source OSS system for ISP, service and content providers. Major features are:

  • Telecom-specific modules
    •    - Network Inventory
    •    - IP Address Management
    •    - VLAN Management
    •    - Service Activation
    •    - Configuration Management
    •    - Fault Management
    •    - DNS Provisioning
    •    - Performance Management
    •    - Peering Management
    •    - Knowledge Base
  • Vendor-agnostic
    •    - Over 30 supported vendors, from CPEs to core MPLS routers
  • Battle-proven
    •    - Used by small and large companies worldwide
  • Leverages mature and proven open-source technologies
    •    - Python
    •    - PostgreSQL
    •    - MongoDB
    •    - Django
    •    - ExtJS
  • Integration
    •    - REST/JSON API
    •    - Python API
  • Established professional community
    •    - DevTeam and Community
  • BSD License

For installing NOC CBSD jail use bsdconfig cbsd and Repository submenu, or just follow this steps:

			% cbsd repo action=get sources=img name=noc

or

			% cbsd repo action=get sources=img name=noc ver=10.1

If you hosted on another ( non 10.1 ) FreeBSD version.

a clean NOC Project installation in the jail will be available for access via http://FQDN
