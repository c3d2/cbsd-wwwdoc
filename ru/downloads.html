CBSDWEB VirtualBox Appliance

Если вы хотите посмотреть CBSD без установки на рабочую систему (CBSDWEB находится на ранней стадии разработки, не для production применения!), вы можете получить ежемесячный development-snapshot образа в VirtualBox. Имейте ввиду, что VirtualBox не поддерживает Nested Virtualization (Nested VTx/VT-d/IOMMU), по этой причине, гипервизоры bhyve и XEN здесь работать не будут - доступны только jail контейнера. Для работы bhyve/XEN устанавливайте CBSD на обычный сервер.

Параметры доступа:

  • - WEB interface: https://IPADDR/ (см. *notes1)
  • - root user unix/shell password: cbsd
  • - cbsd user unix/shell password: cbsd

    • * notes1: Если DHCP в вашей сети нет, отредактируйте файл /etc/rc.conf для статических адресов, заменив строчку:

    	ifconfig_vtnet0="DHCP"

    на:

    	ifconfig_vtnet0="IP/PREFIX"
	defaultrouter="GW"

    например:

    	ifconfig_vtnet0="192.168.0.2/24"
	defaultrouter="192.168.0.1"

    или:

    	ifconfig_vtnet0="192.168.0.2 netmask 255.255.255.0"
	defaultrouter="192.168.0.1"
