FreeBSD: XEN DOM0

Introduction

Starting from CBSD 10.3.0 experimental support of XEN hypervisor has been added

Since CBSD provides only a script wrapper to work with XEN, for their work you need to hold the system configuration in accordance with the documentationXEN FreeBSD Dom0

To create and operate with XEN through the CBSD, use the prefix x for the corresponding commands: xconstruct-tui, xls, xstop, xstart, xremove
